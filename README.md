# Stock Watchlist App

A Flask-based stock watchlist and trading notes application that fetches real-time stock data using the Finnhub API.

## Features

- Search stock tickers
- Real-time stock price lookup
- Add stocks to persistent watchlist
- Add trading notes
- Delete watchlist entries
- SQLite database integration
- Styled frontend using CSS

## Tech Stack

- Python
- Flask
- SQLite
- HTML/CSS
- Finnhub API
